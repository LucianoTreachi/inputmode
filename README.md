# Inputmode

Aquí pruebo el uso del atributo `inputmode` en formularios web, mostrando ejemplos **correctos e incorrectos** de implementación.

El atributo [`inputmode`](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Global_attributes/inputmode) le indica al navegador qué tipo de teclado mostrar en dispositivos móviles, mejorando la **usabilidad** y la **accesibilidad** de los formularios.

## ¿Por qué usar inputmode?

Cuando un usuario completa un campo numérico (como DNI, CBU, CUIL, CVC, número de tarjeta o código postal) desde un dispositivo móvil, el navegador puede mostrar un teclado optimizado si detecta correctamente el tipo de dato esperado.
